## 买卖股票的最佳时机
贪心：当买入价格最低时，出售的价格最高。保存个最小值，每天算最大收益即可。

## 买卖股票的最佳时机2
动规秒了。

贪心：前提要转换下方程：p[a[r] - a[l]] = p[a[l + 1] - a[l] + a[l+2] - a[l+1] + ... + a[r] - a[r-1]]，所以保证每天的收益是正数即可。

## 分发糖果
类似 除自身外数组的乘积 ，对于数组内相邻关系的题，考虑**左右遍历**分别计算定义。

分发糖果的定义包括两部分：
1. rating(i) > rating(i-1)
2. rating(i) > rating(i+1)

左右分别遍历算出解left[i]和解right[i]，取两者最大值即可。
